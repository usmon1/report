# Practise_make_perfect

Описание

Этот каталог содержит учебный проект/практику для отработки навыков работы с базами данных и простых Python-проектов. В проекте есть SQL-скрипты и файл зависимостей для подготовки окружения.

Структура

- .idea/ - конфигурация IDE (игнорировать в VCS при необходимости)
- .venv/ - виртуальное окружение Python (обычно не хранится в репозитории)
- app/ - каталог с приложением (код проекта)
- requirements.txt - зависимости Python
- scripts.sql - SQL-скрипты для создания/заполнения/тестирования базы данных

Требования

- Python 3.8+ (рекомендуется)
- СУБД (например, PostgreSQL или MySQL) для выполнения scripts.sql
- Рекомендуется со��дать виртуальное окружение и установить зависимости из requirements.txt

Установка и запуск

1. Клонируйте репозиторий:
   git clone https://github.com/usmon1/report.git
   cd report/Practise_make_perfect

2. Создайте виртуальное окружение и активируйте его:
   python -m venv .venv
   # Windows
   .venv\Scripts\activate
   # macOS/Linux
   source .venv/bin/activate

3. Установите зависимости:
   pip install -r requirements.txt

4. Просмотрите и выполните SQL-скрипты в scripts.sql в вашей СУБД (выполняйте только в тестовой базе):
   psql -U <user> -d <db> -f scripts.sql   # пример для PostgreSQL

Примечания

- В каталоге может быть виртуальное окружение и IDE-конфигурация — при добавлении в общий репозиторий рекомендуется игнорировать эти папки (.venv, .idea) через .gitignore.
- Перед выполнением scripts.sql убедитесь, что вы понимаете, какие изменения будут внесены в базу данных.
